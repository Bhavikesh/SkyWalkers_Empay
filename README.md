# HRMS Payroll Engine

A modern, high-performance Payroll and HRMS dashboard built with React, Vite, Tailwind CSS, and Supabase.

## Features
- **Dynamic Dashboard**: View monthly payroll statistics, employer costs, and warnings (e.g., missing bank details).
- **Payrun Management**: Process payroll for thousands of employees automatically utilizing powerful Postgres RPC functions.
- **Payslip Generation**: Detailed payslip views with earnings, deductions, and tax breakdowns.
- **Role-Based Access**: Secure Row-Level Security (RLS) integration ensuring tenant isolation.

## Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- Backend/DB: Supabase (PostgreSQL)

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

*This repository is built specifically for the Odoo Hackathon.*
