# EmPay HRMS: Next-Gen Human Capital Management 🚀

**EmPay** is a premium, enterprise-grade Human Resource Management System (HRMS) designed for speed, security, and a seamless user experience. Developed for the **Odoo Hackathon 2026**, it features a sophisticated 4-portal architecture that bridges the gap between employees, managers, and finance.

## 🏛️ Architecture: 100% Frontend-First
EmPay utilizes a specialized **serverless-local architecture**. By leveraging the power of React Context and the Browser’s LocalStorage API, we’ve built a system that is:
- **Instantaneous:** Zero-latency data fetching and mutation.
- **Reliable:** Works offline and persists data across sessions without a backend dependency.
- **Secure:** All organizational data remains within the client environment during the demo.

---

## 🌐 The 4-Portal Ecosystem

### 👑 1. Admin Portal
*Control and Visibility at Scale.*
- **Global Dashboard:** High-level metrics on organization growth and expenditure.
- **User Provisioning:** Onboard new employees and manage global system permissions.
- **Cross-Module Oversight:** Full access to override or audit all other modules.

### 👥 2. Employee Portal
*Empowering the Individual.*
- **Smart Check-In:** Real-time attendance logging from any page.
- **Self-Service Leave:** Request time off and track balances without administrative friction.
- **Personalized History:** View private attendance logs and historical payslips.

### 💼 3. HR Portal
*People Management, Optimized.*
- **Workforce Directory:** Specialized filters for departments, roles, and status.
- **Leave Lifecycle:** Automated workflow for approving or rejecting time-off requests.
- **Talent Monitoring:** Track presence and leave trends to ensure team health.

### 💰 4. Payroll Portal
*Financial Accuracy & Professionalism.*
- **Computation Engine:** Automated calculation of Gross, Net, Allowances, and Deductions.
- **One-Click Payrun:** Process monthly salaries for the entire organization instantly.
- **Document Generation:** Real-time generation of professional PDF payslips.

---

## ✨ Key Features
- **Glassmorphism UI:** A sleek, dark-themed interface built with modern CSS techniques.
- **Auto-Seeding Logic:** The system comes pre-loaded with 30 days of realistic history for a production-ready demo experience.
- **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile workflows.
- **Dynamic Analytics:** Interactive charts providing insights into departmental distribution and salary trends.

---

## 🛠️ Tech Stack
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router & Turbopack)
- **State Engine:** React Context API
- **Persistence:** LocalStorage Sync
- **Icons:** Lucide React & Google Material Symbols
- **Styling:** Modular Vanilla CSS

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bhavikesh/SkyWalkers_Empay.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch the platform:**
   ```bash
   npm run dev
   ```

4. **Demo Access:**
   Open `http://localhost:3000` in your browser.
